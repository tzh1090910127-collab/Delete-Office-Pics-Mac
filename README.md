# Mac 一键删除 Word/Excel 所有图片

一行命令 / 双击运行，批量删除所有 `.docx` `.xlsx` 文件中的：
- 图片（内联+浮动）
- 图表 Chart
- SmartArt
- 页眉页脚图片
- 形状

支持 10000+ 文件，10 秒搞定！

## 使用方法

### 方法1：双击运行（最简单）
1. 下载 [Delete Office Pics.app](https://github.com/你的用户名/Delete-Office-Pics-Mac/releases)
2. 把 Word/Excel 文件拖到图标上 → 自动删除

### 方法2：终端运行
```bash
python3 delete_pics.py
